# LIHO-UBUN (lihoubun)
- Docker ubuntu for api development

# Dependencies
- Package included
```
- docker-composer version 3
- ubuntu 16.07 (nginx, curl)
- mysql 5.7.14
- php 7
- fuelphp 1.8
- sass 3.4.23
- python 2.7.12 & 3.5.2
```

# INIT
- Host file /etc/hosts
```
api-v1.lihoubun.dev
api-v2.lihoubun.dev
```
- Run CLI
```
./help build
./help init
./help up

# Start api v1
./help api v1
```

# Local development
